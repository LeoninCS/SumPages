# internship-skill 使用指南 GitHub Repository

Internship work tracking

自动记录 实习产出， 生成日报和总账。

internship-skill 会在白天记录轻量工作事件，晚上汇总 Git、变更文件和笔记证据，生成日报，并维护一份可搜索、可保存的实习总文档。

用途

日常目标

白天记录 commit、工作树、触达文件和笔记线索，晚上生成可读日报，并把每天内容汇总到 `internship-ledger.md`。

最终价值

实习结束时可以从日期证据中整理成果、影响、技术成长、协作记录和简历 bullet。

第一次配置

克隆仓库并运行安装脚本，安装脚本会把当前仓库链接到 Codex skills 目录。

install 
git clone https://github.com/LeoninCS/internship-skill.git
cd internship-skill
./install.sh
copy config 
cp config.example.json config.json

配置仓库

在 `config.json` 里写实习相关仓库路径。可以直接写路径，也可以维护一个仓库清单文件。

repos 
"repos": [
  "/path/to/frontend",
  "/path/to/backend"
]
repo list 
"repo_lists": [
  "/path/to/internship-repos.txt"
]
environment 
export OPENAI_API_KEY="your-key"

每天如何运行

白天持续记录工作流水

每 15-30 分钟运行一次，把新增 commit、工作树和触达文件事件写入 `events/YYYY-MM-DD.jsonl`。

event capture 
python3 scripts/record_work_events.py --config config.json

晚上生成日报

夜间流水线读取事件流、仓库证据和笔记，生成当天 Markdown summary。

daily summary 
python3 scripts/run_daily_pipeline.py --config config.json

刷新实习总账

把日报、事件、recap 合并到最主要的阅读入口。

ledger 
python3 scripts/build_ledger.py --config config.json

日常主要看这一份文件

~/internship-summaries/internship-ledger.md

实习结束总复盘

按实习起止日期生成总复盘，然后刷新同一份 ledger。

recap 
python3 scripts/build_internship_recap.py \
  --config config.json \
  --start 2026-06-01 \
  --end 2026-08-31

python3 scripts/build_ledger.py --config config.json

产物结构

  • internship-ledger.md：最主要的总文档。
  • YYYY-MM-DD.md：每天晚上生成的 AI 日报。
  • YYYY-MM-DD.evidence.md：原始证据包。
  • YYYY-MM-DD.prompt.txt：用于生成日报的 prompt。
  • events/YYYY-MM-DD.jsonl：白天持续记录的事件流。
  • recaps/YYYY-MM-DD_to_YYYY-MM-DD.md：实习结束总复盘。
  • .cache/：按证据 hash 缓存的 AI 响应。

自动化建议

白天任务

每 15-30 分钟运行 `record_work_events.py`，只做轻量事件追加。

夜间任务

下班前后运行 `run_daily_pipeline.py` 和 `build_ledger.py`，生成日报并刷新总账。
